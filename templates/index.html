Welcome to my API!

Available Routes:

Precipitation: /api/v1.0/precipitation

 This will get all precipitation data from every station recording this type of data.

List of Stations: /api/v1.0/stations

 This will get all data about stations collecting data. You should be able to find station_id, name, lat and long, and its elevation.

Temperature for one year: /api/v1.0/tobs

 This will get all data about temperature readings and a corresponding date.

Temperature stats from the start date yyyy-mm-dd: /api/v1.0/yyyy-mm-dd

 Put in a date and it will get you the average temperature, max temp, and min temp for that date across all stations.

Temperature stat from start to end dates yyyy-mm-dd: /api/v1.0/yyyy-mm-dd/yyyy-mm-dd

 Returns the same data as above but it is average out between the dates provided for start and end
